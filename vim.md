## Movement

| Action                     | Command                |
| :--                        | :--                    |
| Move to end of line        | `$`                    |
| Move to end of line & edit | `A`                    |
| Move to end of file        | `G`                    |
| Format JSON                | :%!python -m json.tool |
| Move one page down         | `^f`                   |
| Move one page up           | `^b`                   |
